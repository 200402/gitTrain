Сокращения
git config --***уровень настроек*** alias.***имя сокращения*** '***комманда(ы) которую сокращают***'

Статус проекта (какие файлы отслеживаются/ отправляются и тп)
git status

Добавить файл в отслеживаемое для коммита (индексация файлов)
git add ***Название файла***

Добавить все файлы в отслеживаемое для коммита
git add *

Коммит с комментарием
git commit -m '***комментарий***'

Коммит с комментарием
git commit 

создание ветви
git checkout -b ***название новой ветки*** ***название ветки от которой будет создаваться ветвь (если не указывать, то будет создана от текущей)

визуальная оболочка
gitk
